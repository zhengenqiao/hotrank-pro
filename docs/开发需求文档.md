# HotRank Pro 开发需求文档

**项目**：跨境电商热销榜单工具  
**版本**：v1.0.0  
**日期**：2026-02-22  
**负责人**：艺涵（主负责人）

---

## 📋 项目概述

### 目标
开发一款跨平台桌面应用，聚合Amazon、SHEIN、TikTok Shop等跨境电商平台的热销榜单数据，帮助用户快速获取市场情报。

### 技术栈
- **框架**：Electron 28 + React 18 + TypeScript
- **UI库**：Ant Design + Tailwind CSS
- **数据可视化**：Recharts
- **数据库**：SQLite (better-sqlite3)
- **爬虫**：Playwright
- **状态管理**：Zustand
- **构建**：Vite + electron-builder
